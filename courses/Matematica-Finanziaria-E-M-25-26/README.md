# Mathematical Methods for Finance 2025/2026

This folder contains the complete course page and all course-specific files.

## Structure

- `index.html`: main course page
- `mathematical-methods-for-finance-calendar.js`: lesson dates and calendar configuration
- `calendar-additions.css`: calendar-specific styling
- `materials/lectures/`: slides and lecture notes
- `materials/exercises/`: exercises and solutions
- `materials/exams/`: examination material
- `materials/other/`: datasets and supplementary resources

Example link from `index.html` to a PDF:

```html
<a href="materials/lectures/lecture-01.pdf" target="_blank">
    Lecture 1
</a>
```
